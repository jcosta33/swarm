# specs/

Your `*.swarm.md` source specs live here — the intent a spec repo authors and reviews, and the thing agents
build from. Each spec is plain Markdown carrying SOL obligations; `example.swarm.md` is a minimal one to copy.

Other intent docs (ADRs, audits, findings, PRDs, RFCs) are normal `type:`-tagged Markdown — keep them
top-level too, wherever suits you. `.agents/` holds only the Swarm tooling (skills, reference cards,
templates, memory).
