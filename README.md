# Swarm — agentic documentation framework

This repository holds **framework prose** (`/docs`) and a **copy-paste scaffold** (`/scaffold`) for adopting Swarm in your own codebase. There is **no CLI or runtime here** — only Markdown artefacts and conventions.

- **Understand:** start at [`docs/README.md`](docs/README.md) (concepts → guides → reference).
- **Install in a project:** follow [`scaffold/README.md`](scaffold/README.md).
- **Design decisions:** see [`docs/adrs/README.md`](docs/adrs/README.md).
