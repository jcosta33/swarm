## TL;DR

Research answers **why & what-if** with traceable lineage to sources. Unlike specs, exploratory dead-ends belong — but must stay clearly labeled as exploratory.

## Divergence: Technical vs UX branches

Technical Researcher stresses reproducible experiments (libraries, benchmarks). Surveyor path emphasises synthesized qualitative signals mapped to UX constraints scaffold encodes differential proof obligations.

Unified skill file still helps maintain parallel structure lowering cognitive load swapping contexts.

## Why explicit promotion of uncertainty

Agents prematurely collapse `[pending]` into faux `[confirmed]` to appear decisive. SKILL mandates visible uncertainty states resisting narrative closure bias — feeds honest Distillation Loss later.

Anti-pattern this fights: covert authority tone without measurement.

Handoff upward always through `spec-writing` / `audit-writing` — research alone never green-lights prod edits ([doc gatekeeper interplay](documentation-gatekeeper.md)).
