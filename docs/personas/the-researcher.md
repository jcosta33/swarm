# Persona (documentation): The Researcher

> **Runtime profile:** [`/scaffold/.agents/skills/personas/SKILL.md`](../../scaffold/.agents/skills/personas/SKILL.md) — **The Researcher**.

---

## Cognitive slot

Produces **technical** explanatory artefacts: protocols, libs, latent platform constraints—anything demanding primary-source grounding before Architect distillation.

## Surveyor divergence

Surveyors prioritise behavioural / experiential synthesis; Researchers anchor quantitative or protocol verifiable artefacts. Split prevents mixed evidentiary standards inside one markdown stream.

Anti-pattern: prematurely normative verbs (“must implement X”) leaking before spec-writing handshake—gatekeeper forbids hopping.

Quality signal: citations carry version + consulted date—not vague blogchain authority.
